★ 함수
- 특정 값을 만들어 내는 수식(코드의 묶음)
- x 값에 따라서, 서로 다른 y 값을 계산하여 반환하는 코드의 묶음
- y = f(x)
- 함수 = function = 기능
- 반환형의 자료형에 따라 함수의 자료형이 결정된다.
- 기본적으로 함수와 매개변수의 이름을 통일하지 않는다.

★ 함수의 정의
- 함수의 형태를 정의한다. 반환 자료형, 함수명, 매개변수를 작성하고 함수가 실행되면 수행할 코드를 {} 안쪽에 작성한다.
- 함수는 마지막에 반환자료형에 맞게 값을 반드시 return해야 한다.

★ 함수의 호출
- 함수를 호출할 때 매개변수에 들어갈 값을 인자 또는 인수라고 부른다.
- 이미 정의되어 있는 함수의 이름을 부르면서(call) 인자값을 전달한다.
- 함수를 호출하면 정의된 내용을 수행하고, 결과값을 호출자리로 반환한다.

★ 함수의 특징
- 자바의 함수는 반드시 반환자료형에 명시된 값을 반환해야 한다.
- 제어문에 의해 분기가 나누어지더라도 반드시 반환값을 처리해 주어야 한다.
- 반복문 안에서 return을 수행하면 반복문과 상관없이 함수가 즉각 종료된다.
- 결과를 return할 때 삼항연산자를 이용하여 값을 결정할 수있다.

★ 문자열 함수
- charAt(i) : i번째 글자를 char형으로 반환하는 함수
- concat(s) : 원본 문자열에 s 문자열 덧셈을 수행하는 함수
- contains(s) : 원본 문자열에 s 가 포함 되어있는지 판별하는 함수
- endWith(s) : 원본 문자열이 s로 끝이나는지 판별하는 함수
- equals(s) : 원본 문자열이 s와 같은 내용인지 판별하는 함수
- indexOf(s) : 원본 문자열에 s 문자열이 몇번째에 있는지 판별하여 반환하는 함수(없을 경우 -1 반환)
- length() : 원본 문자열이 총 몇 글자인지 갯수를 반환하는 함수
