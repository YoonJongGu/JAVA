- nextLine 함수의 경우 엔터 이전까지의 내용을 문자열로 저장한다.

★ 자료형 복습
- 원시 자료형은 데이터를 저장하는 목적으로 사용하며 객체가 아니다.
- 원시 자료형은 함수를 포함하지 않고, 오로지 저장목적으로만 사용한다.
- Wrapper Class, 원시 자료형의 변수를 객체 유형으로 포장하는 클래스
- Wrapper Class의 경우 데이터를 저장하면서 관련된 함수를 포함하는 자료형이다.
- Wrapper Class의 경우 parse 관련 함수를 이용하여 형변환이 가능하다.

★ 연산의 특징
- 정수와 정수 덧셈의 경우 값이 더해진 정수타입이 결과로 나온다. (1 + 2 = 3)
- 문자열과 문자열 덧셈의 경우 문자열과 문자열이 붙은 문자열이 결과로 나온다. (1 + 2 = 12)
- 문자열에 정수를 더하는 경우에 연산이 되는 형태가 아닌 해당 정수가 뒤에 붙어 문자열이 결과로 나온다. (12 + 34 = 1234)
- 원시 자료형에 빈 문자열을 더하는 경우는 그 결과가 문자열 타입으로 나온다.

★ 반복문
- while(조건문)
    -> 조건이 거짓이 되기 전까지 반복하여 수행한다.
    -> 조건이 참인 동안 반복하여 수행한다.
- for(조건식)
    -> 반복에 필요한 변수 설정이 가능하고 조건 설정이 가능하다.
    -> 변수의 연산을 이용하여 반복 통제가 가능하다.
    -> 조건이 참인 동안 반복하여 수행한다.
- 무한 반복을 이용하는 경우 break 함수를 이용하면 해당 반복문을 빠져나올수 있다.
